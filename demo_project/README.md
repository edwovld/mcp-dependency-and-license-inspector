# Демо-проект для MCP «Инспектор зависимостей и лицензий»

Эта папка содержит тестовые манифесты и политику для воспроизводимой проверки всех инструментов MCP без платных сервисов и API-ключей.

- **Не устанавливайте** зависимости (`npm install` / `pip install`) — демо использует только файлы манифестов.
- Содержимое **без секретов** и приватных данных.
- Пошаговый сценарий описан в корневом файле репозитория **DEMO.md**.

Используйте путь к этой папке (`demo_project` или `demo_project/`) в параметре `project_path` при вызове инструментов MCP.
