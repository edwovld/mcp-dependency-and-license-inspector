# mcp-dependency-and-license-inspector
Детерминированный MCP‑сервер «Инспектор зависимостей и лицензий»: сканирование манифестов (npm, Python), проверка лицензий и политик, поиск уязвимостей (OSV, стабы в демо), генерация SBOM (SPDX/CycloneDX). Шесть инструментов: граф зависимостей, скан уязвимостей и лицензий, проверка соответствия политике, рекомендации по заменам.
