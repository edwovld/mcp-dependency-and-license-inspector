# Dependency & License Inspector MCP — краткий питч (для руководства)

**Dependency & License Inspector MCP** — единая точка входа для анализа зависимостей, лицензий и уязвимостей проекта прямо из AI-редакторов (Cursor, Claude Code, VS Code). Один запрос в чате — полная картина по npm и Python: граф зависимостей, CVE, лицензии, проверка корпоративной политики, SBOM. Результат — структурированный отчёт и приоритизированные рекомендации; без ручного запуска npm audit, pip-audit и отдельных инструментов.

**Пример использования:** разработчик перед мержем PR пишет агенту «проверь зависимости и лицензии в этом проекте». MCP сканирует манифесты, вызывает OSV (или стабы в демо-режиме), проверяет политику и возвращает сводку: какие CVE критичны, какие лицензии недопустимы, что заменить. Время аудита — минуты вместо часов.
